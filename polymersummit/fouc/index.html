Polymer app first paint / loading demos

  1. body unresolved - Polymer's <body unresolved> features hides page (white-out) until everything is ready. This is an easy control for FOUC.
    • No FOUC
    • Slow first paint
  2. Async loading - async loading elements is fast, but can cause visual FOUC as elements upgrade.
    • FOUC
    • Fast first paint
  3. App shell concept - shows how the app shell concept can remove FOUC by pre-styling markup before it upgrades.
    • No FOUC
    • Fast first paint
  4. Polymer is="body-bind" - shows how you can create a super fast loading app that and still use Polymer's features (data binding, observers, etc.) in the main page.
    • No FOUC
    • Fast first paint
    • use Polymer features in main page

Note: In the demos, open the console to see first paint times (Chrome only).
